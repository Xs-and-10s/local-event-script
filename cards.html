then / and

Sequential and parallel async composition. The runtime awaits everything — you write none of the Promise plumbing.

click to flip →

fade-in  #logo  400ms ease-out and
slide-up #title 360ms ease-out
then wait 1800ms
then emit splash:dismiss
⑃

match / /match

Pattern-matched branching. First arm wins. Wildcard [_] catches everything else.

click to flip →

match $dismissedCount
  [6] ->
    emit cards:complete
  [_] ->
    broadcast card:dismissed
/match
🛡

guard

Declarative precondition on a command. If the guard fails, execution stops silently — no error, no rescue.

click to flip →

<local-command
  name="card:dismiss"
  guard="$cardsState == 'live'"
  do="` ... `"
>
👁

on-enter / on-exit

IntersectionObserver lifecycle. Add a when guard for one-time reveals. Omit it for bidirectional.

click to flip →

<on-enter
  when="$state == 'hidden'"
  run="emit section:enter"
>
<on-exit
  run="emit section:exit"
>
📡

emit / broadcast

emit dispatches within the LES instance. broadcast bubbles through the full DOM.

click to flip →

emit feed:data-ready
// heard inside this LES

broadcast app:booted
// heard by data-on:app:booted
// anywhere in the document
🧩

use-module

Extend the animation vocabulary. Built-in primitives via type=, custom via src=.

click to flip →

<use-module type="animation">
<use-module src="./spring.js">

// Now spring-in is available:
spring-in .card  500ms
