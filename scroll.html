Language primitives

The full vocabulary

⟳

then / and

Sequential and parallel async. Async-transparent — no await needed.

👁

on-enter

Fires when the host element enters the viewport via IntersectionObserver.

👋

on-exit

Fires when the host exits the viewport. Enables bidirectional choreography.

🛡

when guard

Conditional gate. when="$state == 'hidden'" = reveal once only.

How it works

Wrap. Declare. Animate.

The <local-event-script> wraps its section content so IntersectionObserver has real height to observe.

1

Wrap the section content

Put content inside <local-event-script>. The element becomes the IO target and has real layout height.

2

Declare lifecycle hooks

<on-enter when="$state == 'hidden'"> fires once. Omit the when guard and it fires on every crossing.

3

Handle the event

The handler runs stagger-enter or any primitive. then sequences steps; and runs them in parallel.
