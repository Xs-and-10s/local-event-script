LES primitives in this demo

try / rescue / afterwards
Failable async with cleanup. The backbone of form:submit.
shake [axis noise]
Noise-driven displacement. Simplex x-axis for invalid fields.
on-signal
Reacts when $formResult changes. Routes to succeeded or failed.
slide-in / slide-out
Directional transitions. right→left for forward, left→right for back.
@post (SSE)
Datastar action. Server patches $formResult via SSE response.
afterwards
Always runs after try/rescue. Resets formState to idle.
