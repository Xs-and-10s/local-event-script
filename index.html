Language concepts

then / and
Sequential and parallel async composition. Async-transparent — no await needed.
emit / broadcast
emit dispatches locally. broadcast bubbles through the full DOM.
call / @get
call invokes a local command. @get is a Datastar backend action.
match / /match
Pattern-matched branching. First arm wins. Wildcard [_] as fallback.
try / rescue / afterwards
Error handling with optional cleanup. guard rejection never triggers either clause.
on-enter / on-exit
IntersectionObserver lifecycle. Optional when guard for conditional firing.
on-signal
Reactive Datastar signal watcher. when guard evaluated on every change.
use-module
Import built-in or userland animation primitives. type= for built-in, src= for custom.
